# AssemblyClaw

**The world's smallest AI agent infrastructure. Pure ARM64 assembly.**

Pure ARM64 assembly AI agent for macOS on Apple Silicon.

```
Latest measured (2026-02-23T22:37:59.338Z): 35 KB binary · 4 ms startup (--help) · 1 MB peak RSS
```

## Why

Everyone complains about TypeScript using too much memory. But TS is the fastest
way to iterate and get community hooked. Ship value first, then optimize.

The natural progression:

1. **TypeScript** — ship fast, build community, iterate
2. **Zig** — harden it, single binary, zero overhead
3. **ARM64 Assembly** — absolute minimum, prove the point (AssemblyClaw: 35 KB now, targeting < 32 KB)

This project proves that with the spec in hand, AI coding agents can write and
maintain even assembly. The future is Bend (GPU parallelism), Lean 4 (zero bugs),
and languages we haven't invented yet.

Current measured result:

- Binary: 35 KB (`build/assemblyclaw`, stripped)
- Startup (`--help`): 4 ms
- Peak RSS (`/usr/bin/time -l`): 1 MB

Current targets:

- Binary `< 32 KB`
- Peak RSS `< 128 KB`
- Startup `< 0.1 ms`

## Install

```bash
brew install gunta/assemblyclaw/assemblyclaw
```

Requires macOS on Apple Silicon, Ninja, and Xcode Command Line Tools.

```bash
git clone https://github.com/gunta/AssemblyClaw.git
cd AssemblyClaw
ninja
cp build/assemblyclaw /usr/local/bin/
```

## Architecture

Current implementation details:

- ARM64 assembly in `src/*.s` using AAPCS64 conventions
- NEON SIMD in string hot paths (`_strlen_simd`, `_memcpy_simd`)
- Arena allocator (`mmap`-backed, 64 KB page growth, 16-byte allocation alignment)
- Zero-allocation JSON token scanning helpers in `json.s`
- Config/provider paths pass many values as `ptr+len` slices
- Core runtime calls macOS system APIs via libSystem symbols
- HTTP uses system libcurl loaded lazily at runtime (`dlopen`/`dlsym`)

## The Story

Programming languages will keep evolving. The world needs:

- **Bend** — regular programs running on GPUs automatically
- **Lean 4** — programs without bugs, proven correct

But right now, this project shows that even assembly is within reach
when you have a spec and an AI coding agent. Good times to be alive.
